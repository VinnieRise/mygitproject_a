Сайт включает в себя документацию, построенный с использованием [Contentlayer](https://contentlayer.dev) и [MDX](http://mdxjs.com).

## Характеристики

В комплект поставки входят следующие функции:

1. Написание содержимого с помощью MDX.
2. Преобразование и проверка содержимого с помощью Contentlayer.
3. MDX-компоненты, такие как `<Callout />` и `<Card />`.
4. Поддержка оглавления.
5. Пользовательская навигация с предыдущей и следующей страницами.
6. Красивые блоки кода, использующие `rehype-pretty-code`.
7. Подсветка синтаксиса с использованием `shiki`.
8. Встроенный поиск (_в процессе_).
9. Dark mode (_в процессе_).
